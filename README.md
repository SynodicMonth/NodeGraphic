# NodeGraphic
Node-style ghraphics editor based on Qt

![图片](https://user-images.githubusercontent.com/102399121/167288624-19ebafd2-3f6b-4226-bf92-5bcc7656c19c.png)

## Current support nodes:
 - Input::Image
 - Function::Add
 - Function::Diff
 - Function::LUT
 - Function::Contrast
 - Function::Lightness
 - Function::Saturation
 - Function::Grayscale
 - Function::Threshold
 - Output::Output
